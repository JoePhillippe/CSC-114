## AI Assistance Policy

This repository documents all AI tool assistance used during coursework.

Each assignment folder contains an `AI_LOG.md` file that records:
- What AI tools were used and for what purpose
- What the AI produced
- What I changed, added, or rewrote myself
- What I can explain in my own words

This documentation follows my instructor's guidance on disclosed AI use.
